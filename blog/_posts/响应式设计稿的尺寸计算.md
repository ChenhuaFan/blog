## rem

`rem` 单位相对于 `em`，最大的区别在于 `rem` 是**仅相对**于 `:root` (`html`) 元素的 `font-size` 属性的值来确定的。

```css
html {
    font-size: 12px;
}

p {
    font-size: 1rem; /* 12px */
}

h1 {
    font-size: 2rem; /* 24px */
}
```
