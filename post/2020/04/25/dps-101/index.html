# 数字信号处理 101

课程来自 Coursera, Digital Signal Processing

# 1. DSP

# 1.1 Signal，信号

定义

Description of the evolution of a physical phenomenon.

描述物理现象的变化

e.g.:

  • 天气 -> 温度：我们用温度来描述天气的变化
  • 声音 -> 压力：通过测量声波的压力变化，我们可以来描述声音的变化
  • 声音 -> 磁：如果我们把声音记录在磁带上，那么可以通过磁变化来描述声音的变化
