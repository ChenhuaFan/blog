# 类型、值和变量

JavaScript 中的值的类型：

  • 原始类型
    • 数字，字符串，布尔
    • * null，undefined
  • 对象类型
    • 对象：属性的无序集合。
      • 类：日期 Date 类定义代表日期的对象、正则类 RegExp 定义了正则表达式对象、错误 Error 对象定义各种错误。
    • * 数组：带有编号的值的有序集合。有专属语法和方法
    • * 函数：有与之关联的可执行代码的对象。有专属语法和方法

如果函数用来初始化，（使用 new 操作符）则这个函数被称为构造函数。
